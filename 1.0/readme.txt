LR Product Countdown allows you to set counter in the product pages

== Description ==

This plugin adds a "LR Count Down Settings" section to the admin panel for the plugin settings. you can see the count-down in header of the product page. berfore you need to select checkbox from product page of admin panel and select from date and to date for count-down.
here the steps of select " LR Count Down " 
- login wordpress
- select product tab
- select your product
- select Product CountDown tab in product detail page of admin panel
- now you may see the count down at fort-side of the product page

The main goal of the plugin is add count-down on product detail page you may get coupon code with register mail

For more features Please contact us at [Logicrays](http://www.logicrays.com/contact-us-logicrays-india/)
If you like this plugin Please rate us :)

== Plugin Features ==

- Enable/disable count-down at product detail page of wordpress admin
- you may send the mail with coupon code detail which is selected from "LR Count Down Settings" section
- email subject,email tempalte, button text, email form title, email form descripton, count-down background here are the settings of the plugins


== Installation ==

Important: before of all you have to download and activate WooCommerce plugin, which is mandatory for LR Product Countdown to be working. If WooCommerce plugin is active, you just need to activate the LR Product Countdown plugin.

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `LR Product Countdown` from Plugins page.
